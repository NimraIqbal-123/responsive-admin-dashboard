# Responsive Admin Dashboard

## Overview

Responsive Admin Dashboard is a modern web dashboard template built using HTML5, CSS3, Bootstrap 5, and Chart.js. It provides a clean, responsive, and user-friendly interface suitable for admin panels, analytics dashboards, business management systems, and portfolio demonstrations.

The project showcases responsive design principles, interactive components, data visualization, and modern UI practices using Bootstrap 5.

---

## Features

- Responsive Dashboard Layout
- Fixed Sidebar Navigation
- Mobile Offcanvas Sidebar
- Interactive Line Chart using Chart.js
- Dashboard Overview Section
- Recent Activity Table
- Search Bar
- Export & Share Buttons
- Responsive Bootstrap Grid
- Hover Animations
- Smooth Transitions
- Bootstrap Components
- Modern UI Design
- Cross-Browser Compatible
